# 校园网自动登录工具 🚀

> 开机自动检测网络状态，一键自动登录校园网，远程操控无忧！

## ✨ 功能特点

- ✅ **智能检测**：自动判断是否已联网，避免重复登录
- ✅ **一键登录**：检测到未联网时，自动打开浏览器并按下回车登录
- ✅ **自动清理**：登录成功后自动关闭浏览器标签页，不留痕迹
- ✅ **开机自启**：配合启动文件夹，实现全自动运行
- ✅ **轻量无感**：脚本运行完即退出，不影响电脑性能

## 📁 文件说明

| 文件名 | 作用 |
|--------|------|
| `login.bat` | 主程序：检测网络环境、打开浏览器 |
| `click_enter.vbs` | 辅助脚本：自动按下回车键并关闭网页 |
| `README.md` | 项目说明文档 |

## 🚀 使用方法

### 1. 首次准备
- 将整个文件夹放在 `D:\Automatic Campus Network Connection\`
- **手动打开一次** `http://eportal.hhu.edu.cn` 并登录
- 当浏览器询问“是否保存密码”时，选择 **保存**
- 确保 Edge 浏览器已设置为默认浏览器

### 2. 测试运行
- 双击 `login.bat` 测试效果
- 正常情况：浏览器自动打开 → 3秒后自动回车 → 登录成功 → 网页关闭

### 3. 设置开机自启
- 按 `Win + R`，输入 `shell:startup` 回车
- 右键空白处 → 新建 → 快捷方式
- 浏览找到 `D:\Automatic Campus Network Connection\login.bat`
- 命名为“校园网自动登录”，完成！

## ⚙️ 技术原理

1. **网络检测**：`ping baidu.com` 判断是否已联网
2. **环境检测**：`curl` 访问校园网认证页面，判断是否在校内
3. **自动操作**：VBScript 模拟键盘输入 `{ENTER}` 完成登录
4. **自动清理**：模拟 `Alt+F4` 关闭浏览器标签页

## 📝 自定义修改

如需调整等待时间，修改对应文件中的数字：

- `login.bat`：修改 `timeout /t 3` 中的数字
- `click_enter.vbs`：修改 `WScript.Sleep 3000` 中的毫秒数

## ❓ 常见问题

**Q：为什么浏览器没有自动回车？**  
A：可能是页面加载慢，请增加 `login.bat` 中的等待时间（将3秒改为5秒）。

**Q：开机自启后黑框一闪而过正常吗？**  
A：完全正常！脚本运行完自动关闭，不影响使用。

**Q：可以分享给同学用吗？**  
A：当然可以！直接把整个文件夹复制过去就行。
