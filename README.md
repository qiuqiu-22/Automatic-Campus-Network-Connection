河海大学校园网自动登录工具
开机自动联网，远程操控无忧

功能特点
智能检测：自动判断是否已联网，避免重复登录

一键登录：未联网时自动打开浏览器并模拟回车登录

自动清理：登录成功后自动关闭浏览器标签页

开机自启：支持添加到启动文件夹，实现全自动运行

轻量无感：脚本运行完即退出，不影响系统性能

快速使用
1. 首次准备
将本工具放在任意目录（如 C:\Program Files\...）

手动打开 http://eportal.hhu.edu.cn 并登录一次

在浏览器中保存账号密码

2. 测试运行
双击 Automatic Campus Network Connection.bat

若未联网，脚本将自动打开浏览器 → 模拟回车登录 → 关闭标签页

3. 设置开机自启
运行 install.bat，按提示操作即可

安装程序会自动创建启动快捷方式

常见问题
Q：浏览器没有自动回车？
A：可适当增加 login.bat 中的 timeout 等待时间。

Q：开机自启后黑框一闪而过正常吗？
A：正常，脚本运行完自动退出，不影响使用。
